v1.0.0 — Initial Release

  • General chat with Ambot AI (NVIDIA LLM + local models)
  • AI role personas with installable system prompts
  • Cloud image generation via NVIDIA FLUX.1-schnell + FLUX.2-klein-4b
  • Parallel model execution — both NVIDIA models run simultaneously, first success wins
  • LLM prompt enhancer for better image generation results
  • Document generation (study guides, quizzes, flashcards, summaries, lesson plans)
  • PDF and DOCX export for generated documents
  • Text-to-speech with AI preprocessing and emotion detection
  • On-device memory system with scope-based retrieval
  • Conversation summaries for long-running chats
  • Dual NVIDIA API key rotation (auto-failover on rate-limit)
  • 3 images/day per-user cloud quota (expandable with more API keys)
  • Daily limit tracking with SharedPreferences
  • Brutalist monochrome UI design
  • Dark and light theme support
  • Flutter 3.38.6, Dart 3.10.7, Riverpod state management
